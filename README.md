# Caremonitor_Assignment
## Prerequisites

- Node.js
- Git
- Postman

## Getting started

Follow these steps at the command line:

### 1. Clone and Install

```bash
git clone https://github.com/Amolmore456/caremonitor_api.git
cd caremonitor_api
npm install
```


### 2. Run the solution

```bash
npm run start
```


### 3. Make API Call

```bash
    REQ_TYPE: POST,
    Endpoint: /api/heartbeat
```
